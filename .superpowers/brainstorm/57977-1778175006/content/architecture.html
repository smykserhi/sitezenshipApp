🔄 Request Flow

Dev vs Production

🔧 Development

Vite dev server on :5173
Proxies /api/* → Express on :5000
Hot module replacement enabled

🚀 Production

Express on :5000 only
Serves client/dist as static files
All /api/* routes handled by Express
React Router handled by catch-all

🔐 Auth Flow

Register/Login → JWT token returned → stored in localStorage
Every API request sends Authorization: Bearer <token>
Protected routes check token via middleware before responding
