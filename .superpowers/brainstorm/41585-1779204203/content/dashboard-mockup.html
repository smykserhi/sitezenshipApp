Design notes:
• The toggle is centered in the AppBar so it's always visible and easy to reach.
• N-400 mode uses amber (#f59e0b) instead of indigo/green/purple — a clear visual signal you've switched sets.
• Subtitle text under the greeting updates to clarify which set and how many questions it has.
• Progress stats are completely independent — switching sets never affects the other set's progress.
